### Decisions / notes (M5)

- **Tab counts = backlog totals, not post-filter counts.** The count tells you how many cards
  lack the date; the filter is a temporary lens over the visible list. Documented.
- **Quarter/year cells are compact** (a count badge, not chips) — full chips per day across 3–12
  months would overflow. Click-through is via the week/month ranges.
- **Obsidian owns an open `.base` file** and rewrites it (normalizing/unquoting YAML),
  clobbering external edits to view config. To change view options reliably from the CLI:
  detach all bases leaves first, then edit the file, then open. (Added to the harness lore.)
- **Pre-existing noise:** Obsidian logs `ENOENT … Archive/2026/Task F.md` — a stale workspace
  reference from the M4b archive round-trip (file moved back). Not from plugin code.

## Style pass: look & feel + animations

- Analyzed the whole `styles.src.css` (flat, almost no transitions). Principle applied: animate
  **interactions** (hover/active/drop/focus), never data refreshes — so the M6 flicker-free
  reconciler and the calendar's full re-render aren't undermined (no entrance keyframes on cards
  or calendar).
- **Motion vocabulary:** `--kap-transition` / `--kap-transition-slow` / `--kap-ease` on
  `.kap-root`; a single `@media (prefers-reduced-motion: reduce)` block zeroes them → all motion
  off with no per-rule overrides and **no `!important`** (the only two `!important` strings are
  in comments).
- **Polish:** gear rotates 30° + lifts on hover (presses on active); cards lift `-1px` + deeper
  shadow on hover, `grabbing` + slight scale on active; relationship badges, lane toggle, panel
  toggle, calendar nav/range/tab buttons get hover bg + transitions (+ active press on buttons);
  chips nudge + scale; drop target gets a subtle bg + transition.
- **Fixes/affordances:** `.kap-column { overflow: hidden }` clips the colored header shade to the
  rounded corners; grid day cells get a hover bg (signals the click-to-zoom); **today** is now an
  accent-filled **circle** around the day number (calendar convention); thin scrollbars on all
  scroll regions.
- **Live-verified:** board column corners now round correctly; the today circle renders (27 Jun);
  no console errors; no `:root` leak; 175 tests / lint / build green. Reduced-motion verified by
  construction (durations are variable-driven).
